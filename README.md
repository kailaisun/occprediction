## Environment
- The code is tested on Ubuntu 20.04.2, python 3.8, cuda 11.1.


## Installation
 1. Clone this repository
  ```bash
  git clone https://github.com/kailaisun/occprediction
  ```
  
 2. Install 
  ```bash
  pip install -r requirements.txt
  ```
  

## Train and test
### Transformer

```Bash
python multi_source_transformer-copy.py
```
